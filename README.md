# vue3demo

前端技術： `Vue 3`、`Vuex`、`vue-router`、`pinia`、`vee-validate`

<a href="https://unizalin.github.io/vue3demo/dist/#/" target="_blank">名產買賣平台</a>
- 前台：商品查詢和介紹、購物車功能、下單、結帳資訊、管理員登入 。
- 後台：產品管理（新增 / 修改 / 刪除）、訂單管理、優惠券管理、模擬流程。
